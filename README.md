# NeighbourlyNest 

Follow the instructions below:

1. For first-time user. Before running the init.py, run the requirement.py to install all required packages 
2. Using Database.py, change all the DB login information with your own respective login information. Remember to remove the database = " " 
3. Create the database and also the tables needed
4. Populate Bins and Events data using Bins.py and Events.py
5. Now you should have an active running init.py
